# Semanas 1–2 — Python Fundamentals

**Objetivo:** solidificar os fundamentos de Python que sustentam todas as ferramentas posteriores (Spark, Airflow, dbt são escritos e configurados em Python).

## Arquivos

| Arquivo | Conteúdo |
|---------|----------|
| `ex1_comprehensions_generators.py` | List/dict/set comprehensions, generators vs list comprehension (memória), generator pipeline |

- List, dict e set comprehensions
- Generators e `yield` — lazy evaluation em Python
- `map()`, `filter()`, `reduce()` — paradigma funcional
- Funções puras e imutabilidade
- Type hints básicos e `Optional`
